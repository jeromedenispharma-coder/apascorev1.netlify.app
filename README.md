﻿# APA Score – Web Application



This repository contains the source code for **APA Score**, a web application deployed on Netlify.

Live application: https://apascorev1.netlify.app

---


## Description

APA Score is a web-based tool designed to simplify the calculation and interpretation of the APA score.

The application offers a clean and intuitive interface, allowing rapid use directly from any web browser—without installation.


Key features:
- Simple and user-friendly interface
- Automatic APA score calculation
- Clear display of results
- Fully online access
